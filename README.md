  - Built with **Python**, **Pandas**, **NumPy**.

- **[Analysing-the-Sentiment-of-Twitter-Data-on-the-Bitcoin-by-LSTM](https://github.com/jasper1005/Analysing-the-Sentiment-of-Twitter-Data-on-the-Bitcoin-by-LSTM)**
